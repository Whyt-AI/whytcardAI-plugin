# /verify-visual

Run visual verification on recent UI changes.

## Instructions

Follow the WhytCard visual verification protocol:

1. **Take screenshots** at 3 viewports: mobile (375px), tablet (768px), desktop (1440px)
2. **Check both modes**: light mode and dark mode (if supported)
3. **Evaluate as a user**:
   - Visual hierarchy clear?
   - Typography hierarchy respected?
   - Spacings consistent?
   - Contrast AA minimum?
   - Professional and polished?
4. **Check specific issues**: no overflow, no truncation, no broken layouts, focus indicators visible
5. **Output verdict**: READY or NOT READY with specific file:line issues

Take the screenshots now and evaluate them.
